# android_first_app プロジェクト

## 1,プロジェクトの構成
Android StudioのプロジェクトファイルViewに沿っています。

・manifests ディレクトリ
AndroidManifest.xmlファイルが格納されています。

・java ディレクトリ
パッケージ名ごとに分けられたJavaソースコードファイルが格納されています。ここにはJUnitテストコードも含まれています。

・res ディレクトリ
XMLレイアウト、UI文字列、ビットマップイメージなど、コード以外のすべてのリソースが格納されています。それぞれに対応するサブディレクトリに分けられています。

・Gradle Scripts
プロジェクトのビルド関連の設定ファイルがすべて格納されます。
